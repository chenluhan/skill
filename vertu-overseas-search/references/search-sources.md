# 海外餐厅搜索来源 & 预订平台

## 搜索来源优先级

### 全球通用（首选）

| 来源 | 用途 | 可信度 |
|------|------|--------|
| **Michelin Guide** (guide.michelin.com) | 米其林餐厅查询 | ★★★★★ |
| **TripAdvisor** | 全球最广覆盖，评分 + 评论 | ★★★★ |
| **Google Maps** "Restaurant" + 细分标签 | 营业时间、最新评分最准 | ★★★★ |
| **Eater** (eater.com) | 美食媒体，城市精选，无广告 | ★★★★ |
| **The Infatuation** (theinfatuation.com) | 编辑撰写，真实体验，主要覆盖大城市 | ★★★★ |

### 按地区专属来源

**欧洲**
- **TheFork** (thefork.com)：欧洲最大预订平台，含评分和优惠
- **Yelp**：英国、西班牙有效，法国覆盖弱

**北美**
- **Yelp** (yelp.com)：美国、加拿大最主要
- **OpenTable**：高端餐厅预订首选
- **Resy**：纽约、LA 热门餐厅

**日本 / 亚洲**
- **Tabelog** (en.tabelog.com)：日本餐厅最权威，有英文版
- **Gurunavi**：日本连锁及热门餐厅
- **HungerStation / Zomato**：中东、南亚

**澳大利亚**
- **Broadsheet** (broadsheet.com.au)：本地精选
- **Good Food Guide**：悉尼 / 墨尔本高端

**中东**
- **Zomato**：阿联酋、沙特覆盖好
- **OpenTable**：迪拜高端酒店餐厅

## 预订渠道汇总

| 渠道 | 覆盖地区 | 特点 |
|------|---------|------|
| **OpenTable** | 全球（含中国餐厅） | 高端、商务首选，可选日期查可用性 |
| **TheFork** | 欧洲为主 | 常有折扣，支持中文界面 |
| **Resy** | 美国为主 | 热门抢手餐厅常驻此平台 |
| **Tabelog** | 日本 | 评分系统严格，3.5+ 即为优质 |
| **餐厅官网** | 全球 | 高端餐厅通常官网预订更准确 |
| **电话预订** | 全球 | 最后手段，适合当天或紧急情况 |

---

## 搜索时的广告识别技巧

1. 搜索结果前 3 条通常是付费推广，下拉至自然结果
2. TripAdvisor / Google 评分低于 4.0 谨慎推荐
3. 优先选择有大量真实评价（100 条以上）的餐厅
4. 使用 `site:eater.com [城市]` 或 `site:theinfatuation.com [城市]` 绕过广告

---

## 如果搜索结果不理想

- 换搜索词：加上 `locals recommend`、`hidden gem`、`not tourist trap`
- 用 Michelin Guide 官网直接按城市筛选
- 搜索 `[城市] food blog [年份]`，找本地博主推荐
