# Selection Guide

Use this file only when the user wants help choosing a style rather than naming one directly.

## Fast Mapping

- Premium hardware, luxury services, concierge products:
  `apple`, `bmw`, `ferrari`, `bugatti`, `lamborghini`
- Developer tools, infrastructure, technical dashboards:
  `vercel`, `linear`, `warp`, `supabase`, `posthog`
- Finance, trust, payments, enterprise credibility:
  `stripe`, `wise`, `coinbase`, `mastercard`, `ibm`
- Editorial, storytelling, research-heavy products:
  `notion`, `wired`, `theverge`, `clay`
- Consumer entertainment, media, visual energy:
  `spotify`, `playstation`, `runwayml`, `pinterest`

## How To Recommend

Pick 2-3 candidates, then explain each in terms of user impact:

- What the user will feel: premium, calm, urgent, technical, playful, editorial.
- What the UI will likely change: typography, contrast, density, imagery, button shape, whitespace.
- Why it fits the product's business context instead of being a random style reference.

## Good Defaults

- If the user says "high-end" but not "luxury", start with `apple`.
- If the user says "luxury" or "premium concierge", start with `bmw` or `bugatti`, then contrast with `apple`.
- If the user says "developer-facing", start with `vercel`.
- If the user says "clean SaaS", start with `linear` or `notion`.
- If the user says "financial trust", start with `stripe`.
