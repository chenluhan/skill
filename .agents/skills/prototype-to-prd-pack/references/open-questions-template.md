# Open Questions Template

Use this structure for `05-open-questions.md`.

## Table Format

```markdown
# Open Questions

| ID | Related To | Question | Impact | Owner | Default Assumption |
| --- | --- | --- | --- | --- | --- |
| operator-sla | submit-request | How quickly should an operator pick up a request? | Affects progress state and expectation copy. | PM | Async pickup with no committed SLA in v1. |
```

## Rules

- Keep every question specific.
- State why the answer matters.
- Name the likely decision owner.
- Include a default assumption when it helps unblock packaging.
- Do not duplicate the same unresolved issue in multiple files.
