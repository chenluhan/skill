# skill

Source repository for Codex skills, agents, design specs, and local git automation.

## Structure

- `skills/`: production skill source
- `agents/`: future agent source
- `docs/superpowers/specs/`: design docs
- `scripts/`: repo automation and maintenance scripts
- `notes/`: local scratch space, ignored by git

## Git

- Local changes can be auto-committed by `scripts/git-autosync.sh`
- Remote pushes are manual
